Soy un desarrollador full-stack con enfoque en arquitectura de sistemas, diseño de bases de datos y construcción de productos digitales desde cero. Trabajo con el stack completo: desde interfaces modernas con **Next.js / React** hasta APIs robustas con **Node.js / FastAPI**, pasando por infraestructura con **Docker** y administración de servidores. Me apasiona construir soluciones que realmente resuelvan problemas reales.

- 🚀 Construyendo **[Passion Room](https://lmog.vercel.app/)** — plataforma e-commerce para restaurantes con pagos (MercadoPago/Wompi), integración WhatsApp Business y gestión completa de pedidos.
- 🤖 Desarrollando **IA-GENOMA** — app móvil con IA generativa en tiempo real (React Native / Flutter).
- 🍽️ Finalizando **Software Core Tenis** — sistema de gestión para restaurante/bar en Java Swing + MySQL.
- 🛒 Automatizando dropshipping con **Bot Dropshipping** — investigación de productos y fulfillment.
- 🌐 Construyendo **El Parche** — red social con funcionalidades de split-bill (Python FastAPI + React + PostgreSQL).
- 💡 Explorando **APIs de facturación electrónica** y profundizando en **Ciberseguridad**.
- 📱 Creando contenido técnico en **TikTok** para la comunidad de desarrolladores.

> 🎮 Dato curioso: Cuando no estoy en la terminal, me encontrarás personalizando motos o en un buen match de **Warzone / CoD Mobile**.
